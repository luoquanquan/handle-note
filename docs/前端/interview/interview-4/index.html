&& vs ||

  • 在条件判断中
    • &&: 一假必假
    • ||: 一真必真
  • 在赋值操作中
    • A || B => 如果 A 为真返回 A 的值, 否则返回 B 的值(无论真假)
      • 1 || 2 => 1
      • 0 || false => false
    • A && B => 如果 A 为假返回 A 的值, 否则返回 B 的值
      • 1 && 2 => 2
      • 0 && false => 0
  • 逻辑 && 的优先级高于 ||
    • 0 || 2 && false || 3 => 3
      • 由于 && 优先级高, 所以先执行了 2 && false => false
      • 0 || false || 3
      • 从左向右执行 0 || false => false
      • false || 3 => 3
